# 🔐 CipherVault - Secret Message Encryption App

A premium, futuristic secret message encryption application built with React, featuring real AES encryption, multiple themes, and a stunning glassmorphism UI.

![CipherVault](https://img.shields.io/badge/React-18.2.0-blue)
![Vite](https://img.shields.io/badge/Vite-5.0-purple)
![Tailwind](https://img.shields.io/badge/Tailwind-3.4-cyan)

## ✨ Features

- 🔒 **Real AES Encryption** - Industry-standard encryption using crypto-js
- 🔑 **Password Protection** - Optional password-based encryption for extra security
- 🎨 **5 Premium Themes** - Cyberpunk, Neumorphism, Dark Void, Clean Light, Matrix Green
- 💎 **Glassmorphism Design** - Modern blur effects and semi-transparent cards
- ⚡ **Smooth Animations** - Framer Motion for buttery-smooth transitions
- 📱 **Fully Responsive** - Works perfectly on mobile and desktop
- 💾 **Theme Persistence** - Your theme choice is saved locally
- 📋 **Clipboard Integration** - Easy copy, paste, and share functionality
- 🌟 **Premium UI/UX** - Carefully crafted user experience

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone or download this repository

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

## 🔐 How It Works

1. **Encode Mode**: Enter your message and optional password, then click "Encrypt"
2. **Decode Mode**: Paste the encrypted message and password (if used), then click "Decrypt"
3. **Share**: Copy or share the encrypted message with anyone
4. **Security**: Only those with the correct password can decrypt your message
